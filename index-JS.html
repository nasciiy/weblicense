JavaScriptを学ぶメリット

初心者でも学びやすい

JavaScriptは他の言語よりも学びやすく、 プログラム未経験者でも短期間で資格を取ることが可能です。 また、Webデザイナーとしてある程度経験したあとに、 更なるステップアップにもなるので要注目です。 JavaScriptは、歴史こそ浅いが、大きい役割を持っており、 多くのWebサイトで使用されています。 手軽に組み込めることから初心者でも学びやすく人気があります。

即戦力として雇う企業が多い

JavaScriptはWebサイト開発に重宝されており、 現在もなお求人募集が多いです。大手求人サイトでは900件近い JavaScript技術者を募集しています。 中には未経験でも学びながら仕事ができる環境を用意している企業も見られ、 求人倍率が高い一方で、企業のニーズが毎日上昇しているため人手が足りない状況です。

世界での活躍の場が広がる

JavaScriptはWebサイトに動きをつけることができるため、 多くの仕事で重宝されています。企業が新しいサービスや開発をすれば、 その都度Webサイトやアプリの開発や運営が必要になります。 その中でJavaScriptは世界中で広く使われている言語の一つです。 国内外問わず、在宅でも仕事が可能であることが人気の理由であり、 英語やほかの語学力と組み合わせれば、 世界で活躍する場が広がっていきます。

主な資格

CIW JavaScript Specialist

CIWとは？

CIW(Certified Internet Web Professional)とは、 米国Certification Partner社によって運営されている インターネット技術者の技術者認定資格の名称。 資格だけでなく、公式カリキュラムも提供している。 CIWは、世界70カ国以上で実施され、20万人以上がCIW資格を取得している。

試験は全て英語

国際資格であるため、試験は全て英語で出題されます。 その代わり、言語によって資格認定に差が出ることはありません。 資格は専門分野（職種）ごとによって認定されるため、 専門分野に関するスキルを理解していることの証となる。 このため、世界各国のITスキル標準や他資格とレベル比較が可能である。

CIW JavaScript Specialistについて

題数は50問で、試験時間は75分です。 出題範囲は、JavaScriptの基礎、プログラミング、コード開発だけでなく、 データ処理やオブジェクトモデル、Webサイトへの実装、 データーベースとの連携、デバッグ技法などの範囲が広く、全て英語で出題されます。 合格基準は75％であり、CIWが提供する試験の中で難易度が中級レベルです。

受験料、試験会場と日時

受験料は150米ドルです。試験はPSIオンライン受験で 自宅で受ける事ができる他、ピアソンVUE試験センターで 受験することも可能です。 そのため、特定の試験日時は設けられていません。

HTML5プロフェッショナル認定試験

どのような試験なのか

　

特定非営利活動法人エルピーアイジャパン（LPI-Japan）が 認定している資格であり、HTML5,CSS3,Javascriptなど 最新のマークアップに関する技術力と知識を問う日本国内の試験です。 （日本では今後取得したい資格No.1だとか）

試験内容

　

Webの基礎知識からJavaScript、セキュリティ、 グラフィックアニメーションなどなど、 幅広く出題されます。 認定制度には「レベル１」と「レベル２」の２つのレベルがあり、 それぞれ以下のスキルと知識を備えてるIT技術者であることを認定します。

  • レベル１
    1. HTML5を使ってWebコンテンツを制作することができる。
    2. ユーザビリティ・ビジビリティの高いWebコンテンツを設計・制作することができる。
    3. スマートフォンや車載システムなど、様々なデバイスに対応したコンテンツ制作ができる。
  • レベル２
    1. 動的に動作させて高いユーザビリティを実現するリッチユーザインターフェイスアプリケーションを作成することができる。
    2. マルチデバイスに対応し高パフォーマンスで動作する動的コンテンツを作成することができる。
    3. システム間連携を行いリアルタイムな情報を提供するアプリケーションを開発することができる。
    4. スマートフォンなどでネイティブアプリに近い機能を組み込んだ先端のWebアプリケーションを開発することができる。
    5. APIのセキュリティモデルを理解したうえで開発することができる

受験料、試験会場と日時

受験料はレベル１・レベル２共に15,000円です。 問題数はレベル１は６０問で、レベル２は40～45問であり、 試験時間はともに90分です。試験は、コンピュータベース (CBT) で提供。随時、 指定のテストセンターで受験することが可能である。

受験資格と有効期限

レベル１は、とくに受験資格の条件はなく、どなたでも受験可能です。 レベル２は、レベル１に合格していることが条件になっています。 ここで注意が必要なのは有効期限です。有効期限は５年間なので、 レベル１を合格してから５年以内にレベル２を合格することは必要になります。 また、有意性も５年なので、資格取得してから５年以内に活用しましょう。
